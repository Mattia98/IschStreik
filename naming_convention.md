# Naming convention follow this rules:

## PHP
  * ClassName
  * methodName
  * propertyName
  * function_name (meant for global functions)
  * $variable_name
  * CONSTANT_NAME

[Source](http://programmers.stackexchange.com/a/149321)

## Files
  *	ClassName.class.php
  *	other_scripts.php
  *	a-site.php
  *	css_file.css

## CSS
  * a-class
  * an-id

## gettext
  No convention. (Needs discussion)
